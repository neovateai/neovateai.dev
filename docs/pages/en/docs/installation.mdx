# Installation

## Prerequisites

- [Node.js 18 or newer](https://nodejs.org/en/download/)

## Installation

### macOS, Linux

Install Neovate Code using the following command:

```bash
npm i @neovate/code -g
```

### Windows

<Callout type="warning">
  Windows is not supported yet, we are working on it.
</Callout>

### Verify Installation

After installation, you can verify the installation by running the following command:

```bash
neovate --version
```

## Updates
