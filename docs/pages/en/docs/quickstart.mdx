# Quick Start

Get started with Neovate Code in minutes.

## Prerequisites

- [Node.js 18+](https://nodejs.org/en/download/)
- Terminal access
- A coding project

## Installation

Install globally via npm:

```bash
npm i @neovate/code -g
```

Verify installation:

```bash
neovate --version
```

## Initial Setup

Navigate to your project and start Neovate Code:

```bash
cd /path/to/your/project
neovate
```

On first run, you'll be prompted to configure a provider:

![](https://pic.sorrycc.com/proxy/1758452347948-985315267.png)

1. Type `/login` and select your AI provider
2. Enter your API key

![](https://pic.sorrycc.com/proxy/1758452396763-810257439.png)

3. Type `/model` to choose a model

![](https://pic.sorrycc.com/proxy/1757940619232-448570948.png)

## Start Coding

You're now in interactive mode. Try these commands:

```bash
> explain this project
> create a testcase for @src/components/Button.tsx
> what is the purpose of this project
> analyze the dependencies of this project
```

![](https://pic.sorrycc.com/proxy/1757940861403-266888363.png)

## Review & Commit

After making changes, stage them and get a review:

```bash
git add .
neovate /review
```

Generate and commit with AI assistance:

```bash
neovate commit
```
