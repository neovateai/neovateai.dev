# 快速开始

本指南将帮助您在几分钟内开始使用 Neovate Code。

## 开始之前

- 已安装 [Node.js 18 或更新版本](https://nodejs.org/en/download/)
- 打开终端
- 有一个项目可以使用

## 步骤 1：安装 Neovate Code

使用以下命令安装 Neovate Code：

```bash
npm i @neovate/code -g
```

然后运行 `neovate --version` 来验证安装。

## 步骤 2：配置提供商和选择模型

如果您还没有配置提供商和模型，Neovate Code 在启动时会提示您进行配置。

```bash
neovate
```

![](https://pic.sorrycc.com/proxy/1758452347948-985315267.png)

输入 `/login`，选择一个提供商并输入 API Key。

![](https://pic.sorrycc.com/proxy/1758452396763-810257439.png)

然后输入 `/model` 来选择该提供商的一个模型。

![](https://pic.sorrycc.com/proxy/1757940619232-448570948.png)

## 步骤 3：启动 Neovate Code

打开终端并导航到您的项目，然后运行 `neovate` 来启动 Neovate Code。

```bash
cd /path/to/your/project
neovate
```

您将在交互模式下看到 Neovate Code 提示符。

![](https://pic.sorrycc.com/proxy/1757940861403-266888363.png)

## 步骤 4：运行第一个任务

选择一个任务来运行。

```bash
> explain this project
> create a testcase for @src/components/Button.tsx
> what is the purpose of this project
> analyze the dependencies of this project
```

## 步骤 5：审查更改

任务完成后，git 暂存更改。

```bash
git add .
```

然后使用 `/review` 斜杠命令来审查更改。Neovate Code 将比较更改并向您显示建议，包括性能改进、安全问题、代码风格等。

```bash
neovate /review
```

## 步骤 6：提交更改

审查完成后，使用 `neovate commit` 来生成提交消息、提交并推送。

```bash
neovate commit
```
