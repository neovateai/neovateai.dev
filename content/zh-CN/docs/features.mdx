# 功能特性

Neovate Code 是一个功能丰富的 AI 编程代理。以下是你可以使用的功能。

## 核心功能

- **12 个工具** - 内置工具包括 askUserQuestion、bash、edit、fetch、glob、grep (ripgrep)、ls、read、skill、task、todo 和 write，提供全面的编程辅助。

- **多服务商支持** - 支持 Anthropic、OpenAI、Google Gemini、DeepSeek、Moonshot、GLM、Groq、XAI、OpenRouter、HuggingFace、MiniMax、Cerebras、SiliconFlow、VolcEngine、Poe、Copilot、iFlow AI、AIHubMix、ZenMux、Antigravity，或添加自定义服务商。[了解更多 →](/zh-CN/docs/providers)

- **/model** - 在对话中途切换模型，无需重启。[了解更多 →](/zh-CN/docs/models)

- **会话管理** - 继续、恢复或分叉对话。跨会话保留上下文。

- **压缩** - 自动总结长对话，保持在上下文限制内。

- **Bash 模式** - 使用 `!` 前缀直接运行 shell 命令。

- **图片支持** - 粘贴或拖拽图片到上下文中进行视觉理解。

## 高级功能

- **AGENTS.md** - 项目特定的指令文件，指导代理的行为。[了解更多 →](/zh-CN/docs/rules)

- **斜杠命令** - 可复用的提示模板，用于常见工作流。[了解更多 →](/zh-CN/docs/slash-commands)

- **21 个内置命令** - /clear、/context、/exit、/help、/init、/login、/logout、/mcp、/model、/output-style、/resume、/review、/terminal-setup、/bug、/compact、/status、/add-dir、/spec:brainstorm、/spec:save-design、/spec:write-plan、/spec:execute-plan。

- **技能** - 按需加载的指令文件，用于特定任务。[了解更多 →](/zh-CN/docs/skills)

- **规格驱动** - 通过头脑风暴、计划和执行将想法转化为设计。[了解更多 →](/zh-CN/docs/spec-driven)

- **子代理** - 生成子代理用于并行或委托任务。[了解更多 →](/zh-CN/docs/subagents)

- **插件系统** - 提供生命周期事件、模型解析、工具格式化和自定义斜杠命令的钩子。[了解更多 →](/zh-CN/docs/plugins)

- **SDK** - 将 Neovate Code 嵌入你的应用，实现完全的程序化控制。[了解更多 →](/zh-CN/docs/headless)

- **输出样式** - 自定义响应格式。[了解更多 →](/zh-CN/docs/output-style)

- **MCP** - 模型上下文协议，支持 stdio/SSE/HTTP 传输和异步初始化。[了解更多 →](/zh-CN/docs/mcp)

- **记忆模式** - 使用 `#` 前缀将信息持久化到 AGENTS.md。

- **AI Commit** - 智能 git 提交信息生成。[了解更多 →](/zh-CN/docs/ai-commit)

- **Shell 命令生成器** - 自然语言转 shell 命令。[了解更多 →](/zh-CN/docs/shell-command-generator)

- **会话日志查看器** - 浏览和分析会话日志。[了解更多 →](/zh-CN/docs/session-log-viewer)

- **Neovate Code Desktop** - 原生桌面应用。

- **API Key 轮询** - 轮询多个 API 密钥以管理速率限制。

- **Git Worktree 工作区** - 使用 git worktree 创建隔离的工作区。

- **自动升级** - 自动版本更新。

- **API 代理** - 通过自定义代理端点路由请求。

- **消息队列** - 在代理处理时排队消息。

- **工具审批系统** - 在执行前审查和批准工具调用。

## 输入

- **粘贴文本** - 直接将文本粘贴到聊天中。

- **多行输入** - 使用 Option/Alt/Shift + Enter 输入多行。

- **智能建议** - 按 `@` 或 Tab 访问文件和上下文建议。

- **历史搜索** - 按 Ctrl+R 反向搜索命令历史。

- **历史导航** - 使用上下箭头浏览之前的消息。

- **思考切换** - 按 Ctrl+T 切换思考级别。

- **外部编辑器** - 按 Ctrl+G 在首选的外部编辑器中编辑。
