# Shell 命令生成器

使用 AI 将自然语言转换为 shell 命令，并可选择执行。

## 快速开始

```bash
neovate run "列出当前目录下的所有文件"
```

这将打开交互模式，你可以查看生成的命令并选择操作。

立即执行：

```bash
neovate run --yes "备份 config.json"
```

## 功能特性

- **自然语言转换** - 用中文描述你想做的事，获得准确的 shell 命令
- **交互式操作选择** - 执行、复制、编辑或重新生成
- **命令编辑** - 执行前修改生成的命令
- **提示词编辑** - 优化描述以生成更好的命令
- **自动执行** - 使用 `--yes` 跳过确认

## 选项

| 参数 | 描述 |
|------|------|
| `-m, --model <model>` | 指定使用的模型 |
| `-q, --quiet` | 仅输出生成的命令（无 UI，适用于脚本） |
| `--yes` | 无需确认直接执行命令 |
| `-h, --help` | 显示帮助 |

## 交互式操作

命令生成后，你可以选择以下操作：

| 操作 | 描述 |
|------|------|
| **执行命令** | 立即运行生成的命令 |
| **复制到剪贴板** | 复制命令以便在其他地方使用 |
| **编辑命令** | 执行前修改命令 |
| **编辑提示词并重新生成** | 优化描述并生成新命令 |
| **取消** | 不执行直接退出 |

使用方向键导航，回车键选择。按 Esc 随时取消。

## 示例

查找最近 7 天修改的文件：

```bash
neovate run "查找最近7天修改的所有文件"
```

在代码中搜索模式：

```bash
neovate run "在所有 JavaScript 文件中搜索 TODO 注释"
```

无需确认直接执行：

```bash
neovate run --yes "显示磁盘使用情况"
```

使用特定模型：

```bash
neovate run -m claude-sonnet "压缩当前目录下的所有图片"
```

## 配置

你可以在设置中指定默认模型：

```json
{
  "smallModel": "claude-haiku"
}
```

run 命令默认使用 `smallModel` 以获得更快的响应。更多配置选项请参阅[设置](/docs/settings)。

## 静默模式

使用 `-q` 或 `--quiet` 进行非交互式使用，适合脚本和管道操作：

```bash
neovate run -q "列出所有文件"
```

将输出传递给其他命令：

```bash
neovate run -q "查找大于100MB的文件" | sh
neovate run -q "列出 git 分支" | pbcopy
```

行为：
- 仅将生成的命令输出到 stdout（无交互式 UI）
- 成功时退出代码为 0，错误时为 1
- 需要提供提示词（缺少时会报错）
