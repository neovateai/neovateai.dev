# 会话日志查看器

以交互式 HTML 报告的形式浏览和查看会话日志。

## 快速开始

```bash
neovate log
```

这将打开一个交互式会话选择器，您可以选择要查看的会话。

查看特定日志文件：

```bash
neovate log path/to/session.jsonl
```

## 功能

- **交互式会话浏览器** - 从最近的会话中选择，显示消息数量和时间戳
- **HTML 报告生成** - 从会话日志创建样式化的可浏览 HTML
- **消息检查** - 查看用户消息、助手响应、工具调用和结果
- **请求详情** - 查看每个助手回合的模型、工具、请求/响应数据
- **跨平台** - 在 macOS、Windows 和 Linux 上使用默认浏览器打开

## 选项

| 参数 | 描述 |
|----------|-------------|
| `[file]` | 特定 session.jsonl 文件的路径（可选） |
| `-h, --help` | 显示帮助 |

## 示例

交互式浏览所有会话：

```bash
neovate log
```

查看特定会话文件：

```bash
neovate log ~/.neovate/sessions/abc123/session.jsonl
```

## 输出

生成的 HTML 文件保存到当前目录的 `.log-outputs/` 中，并自动在默认浏览器中打开。报告显示：

- 按对话流程组织的消息
- 工具调用及其输入
- 工具结果及成功/错误状态
- 请求元数据，包括模型和时间戳
